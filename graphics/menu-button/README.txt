A script to generate a complete set of site buttons.
Runs on windows 7 with gimp 2.8 and python 3 installed.

Just run (double click) generate.bat.

original active color:  #f86900
         inactive color:#301c28
barva napisu Menu v mobilni verzi: #cc5500
background color: #d8d8d8

Nepodarilo se dosahnout identicke podoby, jako jsou puvodni rucne generovane png. Je to neco mezi soucasnou
Tahoma a Tahoma Bold. Pokud bude potreba generovat nove buttony, najdi novy vhodny font na soucasnych Windows
a vygeneruj timto skriptem kompletni sadu. Je potreba upravit o jeden px layout (zmena vysky enghlish.png z 19px na 18px).
